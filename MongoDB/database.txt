In MongoDB, data is stored in BSON (Binary JSON) format, which allows for a more dynamic and schema-less approach to data storage.

// This is a sample code snippet to find all users with the job title "Software Engineer" in the "Users" collection of a database.And we can also use findOne() method to find a single document that matches the specified criteria.
db.Users.find({
  JobTile : "Software Engineer"
})

// lte stands for "(lt) for less than or (e) for equal to". 
db.Users.find({
  age: {$lte : 25}
  })

//   gte stands for "(gt) for greater than or (e) for equal to".
  db.Users.find({
  age: {$gte : 25}
  })

  // or operator is used to combine multiple conditions in a query. It allows you to specify multiple criteria, and the query will return documents that match any of the specified conditions.
db.Users.find({
  $or:[{
    JobTitle: "Software Engineer"
  },
  {
    age : {$gte :40}
  }]
})

// and operator is used to combine multiple conditions in a query. It allows you to specify multiple criteria, and the query will return documents that match all of the specified conditions.
db.Users.find({
  $and:[{
    JobTitle: "Software Engineer"
  },
  {
    age : {$gte :20}
  }]
})

// Update function is used to modify existing documents in a collection. It allows you to specify criteria to identify the documents you want to update and the new values you want to set for those documents.
db.Users.updateOne({
  username: "Asharib"
},{
  $set:{
    JobTitle : "Abroad"
  
}})


// We can also use update method to add new field to the existing document in a collection. It allows you to specify criteria to identify the documents you want to update and the new values you want to set for those documents.
db.Users.updateMany({
    JobTitle : "Software Engineer"
},{
  $set:{
  status : "Married"
}})

// WE can delete a document from a collection using deleteOne() method. It allows you to specify criteria to identify the document you want to delete.
db.Users.deleteMany({
  age : {$lte:15}
})
